## Visual Style

**Style:** Luxury Minimal
**Why:** Oh Polly's Lover + Explorer archetype demands a design that foregrounds desire and body confidence without visual noise. Luxury Minimal's generous whitespace, editorial photography dominance, and restrained neutral palette let the garments and the body do the talking — exactly like the SKIMS + Jacquemus reference in the brief.
**Quality to preserve across every section:** Unhurried confidence — every element has room to breathe, nothing competes with the photography, and the site feels like walking into a well-lit fitting room where you already look good.

## Emotional Brief

"This site should feel like getting ready for a night out with your best friend — desire, anticipation, and the confidence that comes from knowing you look incredible — not like scrolling through a catalog of clothes."

## Typography

**Heading font:** Playfair Display — Google Fonts
**Body font:** Raleway — Google Fonts
**Mono font:** Cormorant Italic — Google Fonts (for labels, captions, data)

**Import:**
```css
@import url('https://fonts.googleapis.com/css2?family=Cormorant:ital,wght@1,300;1,400&family=Playfair+Display:ital,wght@0,400;0,500;0,600;0,700;0,800;0,900;1,400;1,500;1,600;1,700;1,800;1,900&family=Raleway:wght@300;400;500;600&display=swap');
```

**Next.js font loading (preferred):**
```tsx
import { Playfair_Display, Raleway, Cormorant } from 'next/font/google';

const playfair = Playfair_Display({
  subsets: ['latin'],
  variable: '--font-display',
  display: 'swap',
});

const raleway = Raleway({
  subsets: ['latin'],
  variable: '--font-sans',
  weight: ['300', '400', '500', '600'],
  display: 'swap',
});

const cormorant = Cormorant({
  subsets: ['latin'],
  variable: '--font-mono',
  style: 'italic',
  weight: ['300', '400'],
  display: 'swap',
});
```

**Scale:**
| Role | Size (clamp) | Weight | Transform |
|---|---|---|---|
| Display / Hero | clamp(3.5rem, 9vw, 9rem) | 300 italic | none — sentence case |
| Section Title | clamp(2rem, 5vw, 5.5rem) | 400 italic | none — sentence case |
| Subheading | clamp(1rem, 1.6vw, 1.3rem) | 500 | none |
| Body | clamp(0.875rem, 1vw, 0.95rem) | 400 | none |
| Label / Caption | 0.6875rem | 400 italic | uppercase, 0.14em tracking |
| Mono Data | 0.8125rem | 300 italic | none |

**Rules:**
- Display headings: always `clamp()` — never fixed px
- Hero headline: Playfair Display Italic, weight 300, line-height 0.95, letter-spacing -0.02em
- `<strong>` inside body paragraphs: renders in --color-accent (champagne rose)
- Never center body copy — left-align always
- Label text: always Cormorant Italic, always uppercase
- Section labels (eyebrows): Cormorant Italic, uppercase, letter-spacing 0.14em, --color-text-muted
- Product names: Raleway 400, --text-body size
- Price text: Cormorant Italic 300, --color-text-muted

## Color System

**Palette:**
| Token | Hex | Use |
|---|---|---|
| --color-bg | #faf8f5 | Page background — warm cream, never pure white |
| --color-bg-dark | #0a0a0a | Dark section backgrounds (hero overlay, email capture) |
| --color-surface | #f2efe9 | Card backgrounds, alternating section backgrounds |
| --color-text | #111111 | Primary text on light backgrounds |
| --color-text-light | #faf8f5 | Primary text on dark backgrounds |
| --color-text-muted | #7a7568 | Secondary text, captions, prices, labels |
| --color-accent | #c4967a | Champagne rose — CTAs, highlights, bold words, hover states |
| --color-accent-dark | #a87a60 | Hover state for accent elements |
| --color-border | rgba(17, 17, 17, 0.08) | Dividers, card edges, section separators |
| --color-border-strong | rgba(17, 17, 17, 0.16) | Active borders, input focus |
| --color-destructive | #c44a4a | Errors, out-of-stock warnings |

**Background strategy:** Alternate between --color-bg (warm cream) and --color-bg-dark (near-black) to create editorial rhythm. The hero and email capture sections are dark. New Arrivals, Editorial Collections, and Bestsellers alternate between --color-bg and --color-surface. Never two consecutive sections with identical backgrounds. The Lookbook section is full-bleed with no background visible (images fill completely).

**Text on dark:** #faf8f5 — passes WCAG AAA contrast on #0a0a0a (ratio 18.5:1)
**Text on accent:** #111111 — passes WCAG AA contrast on #c4967a (ratio 4.6:1)
**Accent on dark:** #c4967a on #0a0a0a passes WCAG AA (ratio 7.2:1)

**RGB channel tokens (for rgba usage):**
```css
--color-bg-rgb: 250, 248, 245;
--color-bg-dark-rgb: 10, 10, 10;
--color-accent-rgb: 196, 150, 122;
```

## Border Radius

| Token | Value | Use |
|---|---|---|
| --radius-sm | 2px | Subtle — inputs, tags, product cards |
| --radius-md | 4px | Buttons, small containers |
| --radius-lg | 8px | Not used — Luxury Minimal avoids rounded containers |
| --radius-pill | 999px | Newsletter input + button combo only |

Luxury Minimal leans sharp. Nearly everything is 2px or 0px radius. Rounded elements feel casual — this brand is not casual.

## Shadow / Elevation

| Token | Value | Use |
|---|---|---|
| --shadow-sm | 0 2px 8px rgba(196, 150, 122, 0.06) | Subtle lift — product card hover |
| --shadow-md | 0 8px 32px rgba(196, 150, 122, 0.08) | Nav scrolled state, dropdowns |
| --shadow-lg | 0 24px 64px rgba(10, 10, 10, 0.12) | Modals, overlays, mobile menu |

Shadow colors use the champagne rose accent hue at low opacity for sm/md, near-black for lg. Never pure black shadows on light backgrounds.

## Motion System

**Motion principle:** "Everything moves like a slow exhale before stepping out — deliberate, unhurried, confident."

| Token | Value | Use |
|---|---|---|
| --ease-out | cubic-bezier(0.16, 1, 0.3, 1) | Most entrances, scroll reveals |
| --ease-gentle | cubic-bezier(0.25, 0.46, 0.45, 0.94) | Image scale transitions, hover states |
| --ease-spring | cubic-bezier(0.34, 1.56, 0.64, 1) | Interactive button press feedback |
| --ease-in-out | cubic-bezier(0.4, 0, 0.2, 1) | Page transitions, nav background |
| --duration-micro | 200ms | Hover states, button feedback |
| --duration-enter | 600ms | Scroll entrance animations |
| --duration-slow | 1000ms | Hero reveals, lookbook clip-path, image wipes |

**prefers-reduced-motion:** All GSAP animations and CSS transitions must check `window.matchMedia('(prefers-reduced-motion: reduce)')`. Disable parallax, marquee motion, entrance transforms, and video autoplay transitions. Show content statically with `opacity: 1` and no transform. Lenis smooth scroll must also be disabled — use native scroll behavior.

## Component Rules

### Buttons
- Primary: solid background in --color-accent (#c4967a), text in --color-text (#111111), Raleway 500
- Secondary / Ghost: transparent, 1px --color-accent border, --color-accent text
- Text link button: no background, no border, --color-text with underline offset, arrow suffix " →"
- Hero CTA: solid --color-text-light background on dark hero, --color-bg-dark text, Raleway 500 uppercase, letter-spacing 0.08em
- Hover: `scale(1.02)` + brightness shift to --color-accent-dark — never color inversion
- Padding: --space-2 horizontal, --space-1 vertical (minimum)
- Never use pure black or pure white as the only CTA color
- Border-radius: --radius-md (4px)

### Cards (Product)
- Background: --color-surface
- No visible border at rest — border appears on hover: 1px solid --color-border
- Hover: image cross-fade to second lifestyle image (opacity transition, --duration-enter)
- No translateY hover lift — this is fashion, not SaaS. The image change IS the hover state.
- Content: left-aligned product name (Raleway), left-aligned price (Cormorant Italic)
- Aspect ratio: 3/4 portrait — standard for fashion photography
- Image: always `object-fit: cover`, never `contain`

### Navigation
- Default: transparent background, positioned over hero
- Text color on hero: --color-text-light (white on dark video/image hero)
- Scrolled (>60px): `background: rgba(var(--color-bg-rgb), 0.95)`, `backdrop-filter: blur(12px)`, text switches to --color-text, `border-bottom: 1px solid var(--color-border)`
- Transition: 300ms --ease-in-out
- Logo: centered, SVG wordmark, max 120px wide
- Links: flanking left (max 5), actions flanking right (search, cart icon linking to us.ohpolly.com)
- Mobile: hamburger icon, full-screen overlay with large Playfair Display links
- Active link: --color-accent underline, 2px offset

### Announcement Bar
- Background: --color-bg-dark (#0a0a0a)
- Text: --color-text-light (#faf8f5), Raleway 400, --text-label size
- Sticky at top, above navigation
- Dismissible with subtle X icon (--color-text-muted)
- Height: 36px
- Content: "Free US shipping over $80 · Duties paid · Shop Now →"

### Images
- Never use `object-fit: contain` in hero, gallery, or product cards — always `cover`
- Alt text required on all content images — no empty alt on product or editorial images
- Decorative lookbook images may use `alt=""`
- Lazy loading on all images below the fold: `loading="lazy"`
- Hero image/video: `priority` / eager loading
- Product cards: 3/4 portrait aspect ratio
- Editorial strip: unequal column widths (1.3fr 0.9fr 0.9fr 1.3fr)
- Lookbook: alternating full-width (16/9) and half-width (4/5)

## Prohibition List

The frontend-builder must NEVER:

1. Use white studio-background product photography as the primary image style — lifestyle photography only on all surfaces
2. Put a "SALE", promotional banner, or discount-first language in the hero — the hero is a campaign, not a promotion
3. Use more than 5 top-level nav items or build a dropdown mega-menu
4. Use a colorful or branded background that competes with the garment photography — the site palette is neutral cream and near-black only
5. Use Inter, Roboto, or any generic sans-serif as the display heading font — Playfair Display is the only heading font
6. Center body paragraphs or product descriptions — left-align always
7. Use identical entrance animations on two consecutive sections
8. Use translateY hover lift on product cards — the image cross-fade IS the hover state
9. Use `display: none` on elements that need to animate — use `opacity: 0` or `visibility: hidden`
10. Skip `prefers-reduced-motion` fallback on any animation
11. Use lorem ipsum or placeholder text anywhere — use real Oh Polly copy in approved brand voice
12. Show pricing in GBP or any currency other than USD
13. Use exclamation marks in headlines or "Hey girl" / "obsessed" / "slay" energy in body copy
14. Use stock photography or AI-generated images — Oh Polly campaign photography only
15. Skip US market signals (shipping, sizing, returns, duties paid) — they are conversion-critical
