# Oryzo.ai Page Topology

## Overview
Satirical product landing page for "ORYZO" — the world's most unnecessarily sophisticated cork coaster.
Built with Astro. ~56,700px total scroll height. Each section is sticky/pinned for extended scroll.

## Color Palette
- `--color-white: #ffedd7` (warm cream — primary text/bg)
- `--color-black: #100904` (near-black brown — primary dark)
- `--color-orange: #dc5000`
- `--color-grey-brown: #6c5f51`
- `--color-green: #445231`
- `--color-green-light: #5d6c49`
- `--color-green-lightest: #f6e0c6`
- `--color-brown: #382416`

## Fonts
- Display: `halyard-display-variable` → substitute: `Plus Jakarta Sans` (variable weight)
- Text: `halyard-text-variable` → same substitute
